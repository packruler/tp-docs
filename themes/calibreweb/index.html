    Setup¶

    This is a "re-skin" of the awesome CaliBlur theme made by leram84 and hexeth.

    Go to Admin -> Edit Basic Configuration -> Server Configuration -> And add theme-park.dev to Trusted Hosts and click Save

    Set theme to CaliBlur in UI Configuration -> View Configuration -> Theme -> CaliBlur -> Submit
